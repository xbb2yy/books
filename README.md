# 技术书籍
自己平时阅读的一些书籍，基本都购买了纸质版，电子版空闲时翻翻。主要是Java技术栈，也会看一些python，c方面的书籍，平时也看看机器学习的文章。我倾向于一个领域只看一本书籍，而且要找经典的看，英语基础好的尽量看原版。

:apple: 标识为必至少看一遍的5分书籍
